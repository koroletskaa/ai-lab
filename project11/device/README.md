Пристрій ESP32 
Зчитує серцевий ритм через PulseSensor

Обчислює BPM

Показує пульс на OLED-дисплеї

Передає дані на сервер через Wi-Fi

Фільтрує шум і виявляє піки серцевого ритму

Пристрій ESP32 з датчиком PulseSensor зчитує пульс користувача, обробляє сигнал і виводить BPM на OLED-дисплей. Кожні кілька секунд він надсилає дані на бекенд через Wi-Fi у форматі JSON, де вони зберігаються та синхронізуються з емоційними записами користувача. Пристрій забезпечує постійний моніторинг серцевого ритму та дозволяє отримувати об'єктивні фізіологічні дані для аналітики та звітів на сайті.

## Встановлення

1. Встановіть Arduino IDE та платформу ESP32:
   - Додайте URL до менеджера плат: https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
   - Встановіть платформу ESP32 через менеджер плат

2. Встановіть необхідні бібліотеки:
   - U8g2 (для OLED дисплею): Tools -> Manage Libraries -> пошук "U8g2"
   - ArduinoJson (для JSON): Tools -> Manage Libraries -> пошук "ArduinoJson"

3. Налаштуйте код:
   - Відкрийте `pulse_tracker.ino`
   - Змініть `ssid`, `password` та `serverURL` на свої значення
   - Для ESP32-C3 можливо знадобиться змінити піни SDA/SCL для OLED

4. Завантажте код на плату ESP32-C3

## Підключення

- Pulse Sensor: підключіть до піна A0 (або змініть PULSE_SENSOR_PIN)
- OLED дисплей: SDA та SCL піни (для ESP32-C3 зазвичай 21/22)
- Питание: через USB або акумулятор через зарядний модуль TP4056
