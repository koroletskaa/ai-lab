В цій папці буде знаходитись код для:

 - відображення пульсу, отриманого з пристрою ESP32
 - Міні-графік пульсу за останні години, який постійно оновлюється.
 - Додавання емоцій у форматі «емоція + інтенсивність + опис».
 - Журнал емоцій із фільтрами та зручним переглядом історії.
 - Інтерактивні графіки пульсу й емоцій для аналізу стану.
 - Автоматичні звіти, які можна переглядати або завантажувати.
 - Статус підключення пристрою та індикатор передачі даних.
