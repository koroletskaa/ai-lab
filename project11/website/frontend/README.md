В цій папці буде знаходитись код для:

 - відображення пульсу, отриманого з пристрою ESP32
 - Міні-графік пульсу за останні години, який постійно оновлюється.
 - Додавання емоцій у форматі «емоція + інтенсивність + опис».
 - Журнал емоцій із фільтрами та зручним переглядом історії.
 - Інтерактивні графіки пульсу й емоцій для аналізу стану.
 - Автоматичні звіти, які можна переглядати або завантажувати.
 - Статус підключення пристрою та індикатор передачі даних.

## Запуск

1. Відкрийте `index.html` у веб-браузері або запустіть локальний сервер:

```bash
# Python 3
python -m http.server 8080

# Node.js (якщо встановлено)
npx http-server -p 8080
```

2. Відкрийте браузер і перейдіть на `http://localhost:8080`

3. Переконайтеся, що backend сервер запущено на `http://localhost:8000`

## Налаштування

Якщо backend працює на іншому порту або адресі, відредагуйте `API_BASE_URL` у файлі `app.js`:

```javascript
const API_BASE_URL = 'http://your-backend-url:8000/api';
```

## Функціонал

- **Моніторинг пульсу в реальному часі**: Оновлення кожні 5 секунд
- **Додавання емоційних записів**: Швидка форма з обранням емоції, інтенсивності та опису
- **Інтерактивні графіки**: 
  - Динаміка пульсу
  - Розподіл емоцій
  - Кореляція між емоціями та пульсом
- **Журнал записів**: Історія з можливістю фільтрації
- **Статистика**: Середні показники за період
- **Звіти**: Автоматична генерація звітів для психотерапевта
